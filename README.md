# Digital-literacy-project
Digital Literacy Project – CSE0001

This repository contains my Digital Literacy project completed during my first year of B.Tech in Cyber Security at VIT Bhopal. The main aim of this project was to understand how to use digital technologies in a safe, responsible, and effective way.

Throughout the project, I worked on multiple tasks that helped me explore different aspects of digital literacy. I created an infographic to spread awareness about online safety and digital practices, which improved my creativity and design skills. I also built my digital portfolio using platforms like GitHub and LinkedIn, which helped me understand how to present myself professionally online.

In addition, I practiced basic coding on HackerRank and created a Google Form quiz to learn how digital tools can be used for learning and data collection. I also learned how to write professional emails and understood the importance of proper communication in academic and professional settings.

The project also included a cybercrime case study and a safety checklist, which made me more aware of online threats and how to avoid them. Overall, this project was a great learning experience that improved my technical skills and digital awareness.
